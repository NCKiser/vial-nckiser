Colt45, made by rpiguy, programmed by NoahK

Keyboard Maintainer: QMK Community  
Hardware Supported: Colt45, ATmega32U4 "Pro Micro"  
Hardware Availability: [KiserDesigns.com](http://kiserdesigns.com/)

Copy this folder named "colt45" into your "qmk_firmware/keyboards/" directory.
(this file will be "~/qmk_firmware/keyboards/colt45/readme.md")

Compile QMK for this keyboard:
    qmk compile -kb colt45 -km default
